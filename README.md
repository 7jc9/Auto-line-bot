## important note ⚠️
Before installing the bot, Go to Shell And Type This `npm i discord.js@12.5.3`
## Setup bot :

1. Create bot [Click here](https://discord.com/developers/applications)


2. Create `.env` file
3. Type this :
```
token=<token bot>
prefix=<prefix bot>
```
